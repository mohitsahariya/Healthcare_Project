
# 🚀 Insurance Cost Analytics Dashboard

## 🔍 Overview
This project analyzes an insurance dataset to uncover key factors driving medical charges. 
It includes interactive dashboards, exploratory data analysis, and a predictive model.

## 🎯 Objectives
- Identify cost drivers (age, BMI, smoking, etc.)
- Build interactive dashboards
- Create predictive ML model

## 🛠️ Tech Stack
- Python (Pandas, Plotly, Scikit-learn)
- Streamlit
- Jupyter Notebook

## 📊 Key Insights
- Smokers incur significantly higher charges
- BMI and age strongly influence costs
- Region has minimal impact
